# @ckapp/rxjs-node-fs

This package contains [RxJS][dep-rxjs-npm-url] observable/operators for the `fs` module of `node.js`.

## Installation and Usage

### ES6 via npm

```sh
npm i @ckapp/rxjs-node-fs
```

## Whats in there?

### Observables

- `appendFile`
- `readDir`
- `readFile`
- `rename`
- `stat`
- `watch`
- `writeFile`
